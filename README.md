# 💰 FinTech Services — Web Application

A comprehensive fintech web application providing government schemes information, financial consultancy, budgeting tools, and loan management features — built with Express.js, EJS, and Tailwind CSS.

## ✨ Features

- 🏛️ **Government Schemes** — Browse & search schemes for rural students, businesswomen, and farmers
- 👩‍💼 **Financial Consultancy** — Book appointments with financial experts
- 📊 **Budget Planning** — Track expenses & manage budgets with visual charts
- 🏦 **Loan Management** — Compare loan options & calculate EMIs

---

## 🛠️ Tech Stack

| Technology | Usage |
|-----------|-------|
| Node.js | Runtime environment |
| Express.js | Web application framework |
| EJS | Server-side templating engine |
| Tailwind CSS | Utility-first styling |
| Chart.js | Visual budget charts & reports |

## ⚙️ Prerequisites

- Node.js `v14` or higher
- npm `v6` or higher

---

## 🚀 Getting Started

```bash
# 1. Clone the repository
git clone https://github.com/MitaliRakholiya12/fintech-services.git
cd fintech-services

# 2. Install dependencies
npm install

# 3. Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🔍 Features In Detail

### 🏛️ Government Schemes
- Search and filter by category and state
- Detailed scheme information with required documents

### 👩‍💼 Financial Consultancy

- Real-time appointment booking & availability

- Category-wise visual analysis with Chart.js
- Export functionality for reports

### 🏦 Loan Management

- Side-by-side loan comparison
- Complete application process guidance
